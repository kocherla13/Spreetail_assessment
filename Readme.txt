required: Python 3.0 and above installed
IDE:  Pycharm (preferred) or any IDE

To run the application in CLI

Please go to the app folder and open CLI by typing cmd at the top of the folder

To run application please type:
  Python Multivalued_Dictionary.py

To run Test cases please go to Test_case Folder and type:
  Python UnitTest_Multivalued_Dictionary.py
  Python IntegrationTest_Multivalued_Dictionary.py
